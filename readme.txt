This release is to upgrade existing APACSVNA to APACSVNA 5.30 version.

1) Upgrade existing wildfly and Keycloak databases using mysql scripts.
2) Upgrade existing ApacheDS server by running the scripts provided.
3) Upgrade of Wildfly Server.
4) Upgrade of keycloak server.

Keep a copy of APACSVNA and databases(wildfly and keycloak) before upgrading the existing APACSVNA server. Copy the folder 'upgrade-5.21-5.30' 
to currently installed APACSVNA folder. Follow the steps given in APACSVNA5.30_upgrade.doc inside upgrade-5.21-5.30 folder.
